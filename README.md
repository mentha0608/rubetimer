# Rubetimer

ブラウザでそのまま使える高精度タイマーです。  
音声ガイド付きでタイミング管理をサポートします。

---

## ▶ 使い方
1. 「吹き出し」に合わせてボタンを押してスタート  
2. 吹き出しに合わせられなかった場合、１回目の爆発にあわせて「１」を押してください
3. ダメなときは２回目の爆発までリカバリ可能

## ▶ 主な機能
- 高精度タイマー（performance.now + requestAnimationFrame）
- 事前読み上げ（7秒 / 10秒切替）
- 次案内のON / OFF設定
- 音量調整（VOICE / SE）
