# Rubetimer

ブラウザでそのまま使える高精度タイマーです。  
音声ガイド付きで、タイミング管理を直感的にサポートします。

---

## ▶ 使い方
1. ボタンは吹き出しが出た瞬間に押す想定です。  
2. ルベランギスの腕が1周したのを目安に押す程度です。  
3. 少し遅れたと感じた場合は「-0.5」を使用します。  
　「1」「2」ボタンは途中補正用です。  
4. 設定から事前読み上げ秒数を 7秒 / 10秒 で選べます。  
5. 次案内が不要な場合は、設定から再生をOFFにできます。

## ▶ 主な機能
- 高精度タイマー（performance.now + requestAnimationFrame）
- 床出現タイミング基準の進行
- 事前読み上げ（7秒 / 10秒切替）
- 次案内のON / OFF設定
- 音量調整（VOICE / SE）
- 途中補正用の「1」「2」ボタン

---

## ▶ 現在の基準値

### サークル
- 初回床まで：約 5.7秒
- ループLap：23.75 / 20.75 / 23.25 / 21.00 / 25.00 / 20.75

### グランド
- 初回床まで：約 6.8秒
- ループLap：22.00 / 24.50 / 22.25 / 24.00 / 22.75 / 23.50 / 23.25 / 23.50 / 23.25 / 23.00 / 23.75 / 22.50

## ▶ 更新履歴

### v2.30 
- Howler.js による音声再生基盤へ移行
- Safari / iPhone 向けの音声再生安定化を目的に再生処理を刷新
- Audio の自前管理から Howler 音源管理へ変更
- 初回ユーザー操作で Howler の再生基盤を初期化する処理を追加
- VOICE / SE / テスト再生を Howler ベースへ統一

### v2.21
- グランドの「網・線上・外」音声が再生されない不具合を修正
- 設定の保存 / 読み込み処理を修正

### v2.20
- タイマー基準を爆発基準から床出現基準へ変更
- サークル / グランドのLap値を実測ベースで更新
- 初回床までの初期値を調整
- 次案内タイミングを調整

### v2.11
- 設定UIを調整
- 事前読み上げ秒数を選択式に変更
- 次案内の再生ON/OFF設定を追加
